# Playwright Automation Project

This repository contains automated UI tests built with Playwright and TypeScript using the Page Object Model (POM) pattern.

## 🛠 Prerequisites

Before running the tests, ensure you have Node.js installed (node -v)
## Getting Started

1. **Clone the repository:**
   git clone <repository-url>
   cd <project-folder>

Install dependencies:
npm install


Install playwright:
npx playwright install



#Environment Configuration
The project uses dotenv to manage environment variables. To run tests, you need to create a .env file in the root directory.
Copy the template from the env-example file in the core folder.
